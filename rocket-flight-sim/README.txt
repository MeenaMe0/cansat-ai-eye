Instructions
>> .ork (Openrocket) file *preferred*
	Use with openrocket (free download), 4m/s wind simulation is mostly
	preferred. General data - Velocity, Altitude, Mach No., Stability, etc.
	to see data go to flight simulations tab, then click plot and export
	.csv export available
>> .CDX1 (RASAero) file
	Use with rasaero ii (free download - windows only)
	Before every use go to file -> select motor file and choose the .eng file
	attached in this zip. Run the simulation in Flight simulations tab. Right
	click and run all. To see detailed numbers go to Aero plots tab.
	.csv export available
>> .ipynb (RocketPy) file *preferred
	Use with any python interpreter (vscode recommended)
	The file attached already have outputs. If re-running the sim, make sure
	all line including paths is corrected to local location, essential files
	are included. (thrust source, power on-off drag csv)
